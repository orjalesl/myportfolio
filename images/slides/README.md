# Slide images (customer-growth-2.html)

Export each PowerPoint slide as an image and drop it here using these exact
file names. Until a file exists, the page shows a labeled dashed placeholder.

- `overview.png`
- `architecture.png`
- `workflow.png`
- `models.png`
- `results.png`
- `dashboard.png`
- `lessons.png`

Tips
- Use a 16:9 export (the viewer frame is 16:9). PNG or JPG both work — if you
  use `.jpg`, update the matching `data-src` in `projects/customer-growth-2.html`.
- Export at ~1920×1080 for a crisp full-screen view.
- To add/rename/reorder slides, edit the `<button class="cs-tab" …>` list in
  `projects/customer-growth-2.html` (each tab's `data-src` = its image path,
  `data-caption` = its title). The viewer and lightbox pick them up automatically.
